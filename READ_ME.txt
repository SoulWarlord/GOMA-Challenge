Desenvolvimento de uma Aplicação Web - GOMA Development

2 páginas web que permite:
	- 1º página: Registar um cliente através de um formulário. É indicado ao utilizador se a sua submissão foi
	efectuada com sucesso ou se deve reescrever a informação e tentar novamente a submissão dos dados
	- 2ª página: É listado por ordem descendente de ID da base de dados, todos os clientes registados na plataforma.
	Existe a opção "ULTIMOS 3 REGISTOS" que permite visualizar os últimos 3 registos de clientes, por ordem
	ascendente de nome. (NOTA: Achei por bem adicionar um filtro para voltar a visualizar a lista completa novamente
	com o nome "VOLTAR AO REGISTO COMPLETO")

Desenvolvido com PHP, HTML, CSS e MySQL
Utilizei Bootstrap em alguns estilos para melhor aproximação ao design do layout proposto.
O ficheiro SQL que permite construir a base de dados e respectiva tabela encontra-se na pasta "database sql".
